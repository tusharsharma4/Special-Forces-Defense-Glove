# Special Forces Defense Glove

## Overview
The Special Forces Defense Glove is a hardware-based defense project designed for soldiers and security personnel. The glove is developed to support hand-to-hand combat situations and improve tactical response during close-range operations.

This project combines electronic components and wearable technology to create a compact and practical defense system.

---

## Features

- Wearable defense glove design
- Compact and lightweight structure
- Designed for close combat assistance
- Electronic activation mechanism
- Rechargeable power source
- Hardware-based implementation
- Tactical and military-inspired concept

---

## Components Used

- Rechargeable Battery
- Battery Holder
- Copper Coil
- Induction Coil
- Switch
- Jumper Wires
- Protective Glove Base
- LEDs (Status Indicators)
- Electronic Connectors
- Insulation Materials

---

## Working Principle

The glove works using an electronic activation mechanism integrated inside a wearable glove structure. When activated using the control switch, the power source energizes the internal circuitry and activates the defense module.

The compact design allows mobility and ease of operation during tactical situations. LEDs are used for indication and system monitoring.
